Introduction to HTML

HTML(HyperText MarkupLanguge) is the code that is used to structure a webpage and it's content.For example, content could be structured within a set of paragraphs,a list of bulleted points or using images and data tables.

Ps:It was invented by Physcist Tim Berners-Lee,the first version was published in 1993.

So what is HTML?

HTML is a markup language that defines the structure of your content.It consists of a series of elements which you use to enclose,or wrap diffent parts of the content to make it appear or act a certain way.For example,if we wanted a line to stand by itself,we could specify that it is a paragraph by enclosing it in paragraph tags.

Anatomy of an HTML element

The main parts of an HTML element are as follows:

  1. opening tag: This consists of the name of the element(e.g),wrapped in opening and closing angle brackets.This states where the element begins or starts to take effect.
  2. The closing tag:This is the same as the opening tag,except that it includes a forward slash before the element name.This states where the element ends-in this case where the paragraph ends.

Summary

HTML is a markup language used to describe the structure and content of a webpage,it says what comes on a webpage.An HTML document is made up of series of elements.Some elements we covered in this lesson include;
